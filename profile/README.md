# Trabalho PSPD

Este projeto foi desenvolvido como parte da disciplina **Programação para Sistemas Paralelos e Distribuídos (PSPD)**, do curso de **Engenharia de Software** da **Universidade de Brasília (UnB)**, ministrada pelo **Prof. Dr. Fernando W. Cruz**.

## Repositórios

* [grpc-communication-examples](https://github.com/Trabalho-PSPD/grpc-communication-examples): demonstra, com exemplos de aplicações, os tipos de comunicação suportados pelo gRPC, incluindo `unary calls`, `server-streaming calls`, `client-streaming calls` e `bidirectional streaming calls`.
* [app-grpc](https://github.com/Trabalho-PSPD/app-grpc): aplicação desenvolvida com arquitetura baseada em microserviços utilizando gRPC para a comunicação interna entre o API Gateway e os serviços.
* [app-rest](https://github.com/Trabalho-PSPD/app-rest): aplicação equivalente desenvolvida utilizando comunicação REST/JSON, permitindo a comparação de desempenho e funcionamento com a versão em gRPC.
